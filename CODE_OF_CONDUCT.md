# 貢献者行動規範（Contributor Covenant）

## 私たちの誓い

私たちメンバー、貢献者、およびリーダーは、年齢・体格・目に見える / 見えない障害・民族・
性的特徴・性自認と性表現・経験の度合い・教育・社会経済的地位・国籍・容姿・人種・宗教・
性的指向にかかわらず、誰にとってもハラスメントのないコミュニティへの参加を実現することを誓います。

私たちは、オープンで歓迎的・多様性に富み・包摂的で健全なコミュニティの実現に資する形で
行動し、交流することを誓います。

## 私たちの基準

コミュニティにとって望ましい行動の例:

- 他者への共感と思いやりを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを行い、また快く受け入れる
- 自分の過ちの影響を受けた人に責任を持って謝罪し、その経験から学ぶ
- 個人だけでなくコミュニティ全体にとって最善なことに目を向ける

許容されない行動の例:

- 性的な言葉・表現の使用、および性的な注目や誘いかけ
- 煽り、侮辱、中傷的なコメント、個人的・政治的な攻撃
- 公然または個人的なハラスメント
- 明示的な許可なく、他者の物理的住所やメールアドレスなどの私的情報を公開すること
- 職業上の場にふさわしくないと合理的に判断されるその他の行為

## 執行責任

コミュニティのリーダーは、許容される行動の基準を明確にし執行する責任を負い、
不適切・脅威的・攻撃的または有害と判断した行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

本行動規範は、コミュニティのすべての空間に適用され、また個人がコミュニティを公式に
代表する場合には公共の空間にも適用されます。

## 執行（報告方法）

虐待的・嫌がらせ的・その他許容されない行動を目撃した場合は、メンテナ（GitHub: **@satoh-y-0323**）まで
ご報告ください。機微な内容の場合は、リポジトリの **Security** タブからの非公開報告も利用できます。
すべての報告は速やかかつ公正に検討・調査され、リーダーは報告者のプライバシーと安全を尊重します。

## 帰属

本行動規範は [Contributor Covenant](https://www.contributor-covenant.org) version 2.1
（https://www.contributor-covenant.org/version/2/1/code_of_conduct.html ）を基にしています。
